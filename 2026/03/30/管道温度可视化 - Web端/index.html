管道温度可视化项目(二)：红外相机温度映射-Web端

功能描述

本次功能流程概括，红外相机定点拍摄管道图片，提取图片中心区域温度，将中心区域的最高温度或者平均温度以Json数组的形式接收，我收到Json数组的数据后，通过解析数据得到各点位温度，将温度信息映射到管道，实现温度映射。
功能包含–Web端：
1.STL实体模型加载与解析
2.自定义颜色映射表，实现从20度(灰色)到100度(红色)的温度映射
3.基于IDW(反距离权重)空间插值算法，实现定点之间的热量扩散效果
4.基于Raycasting(射线投射)的3D鼠标拾取器，鼠标悬停在管道任意位置时，系统可穿透空间，精准命中管道物理表面，并在屏幕上实时显示该点三维坐标(X,Y,Z)及当前温度，温度超过一定值触发视觉报警
5.每5s异步轮询最新Json数据，降低网络带宽消耗，每1.5s独立执行物理热力计算和GPU画面更新，降低屏幕卡顿

##实现流程

由于项目需要在Web端呈现，那么首先我们要下载相关软件，这里我们下载一个VS Code和node.js，在VS Code中新建工程，我们之前在VS Studio里面了解到了VTK的可视化管线，那么在JavaScript中的流程也是一样的，不过在JavaScript中使用的是VTK.js,使用前我们先引入vtk.js模块

// 1. 引入 vtk.js 的核心模块
import vtkPointPicker from ‘@kitware/vtk.js/Rendering/Core/PointPicker’;
import ‘@kitware/vtk.js/Rendering/Profiles/Geometry’;
import vtkFullScreenRenderWindow from ‘@kitware/vtk.js/Rendering/Misc/FullScreenRenderWindow’;
import vtkSTLReader from ‘@kitware/vtk.js/IO/Geometry/STLReader’;
import vtkMapper from ‘@kitware/vtk.js/Rendering/Core/Mapper’;
import vtkActor from ‘@kitware/vtk.js/Rendering/Core/Actor’;
import vtkDataArray from ‘@kitware/vtk.js/Common/Core/DataArray’;
import vtkColorTransferFunction from ‘@kitware/vtk.js/Rendering/Core/ColorTransferFunction’;
import vtkCellPicker from ‘@kitware/vtk.js/Rendering/Core/CellPicker’;

然后初始化渲染窗口，并配置背景色为黑色，

// 2. 初始化渲染窗口
// vtk.js 提供了一个超级方便的全屏渲染器，自带相机控制
//创建可视化场景实例
const fullScreenRenderer = vtkFullScreenRenderWindow.newInstance({
background: [0.1, 0.1, 0.1], // 设置背景色为黑色
});
const renderer = fullScreenRenderer.getRenderer();
const renderWindow = fullScreenRenderer.getRenderWindow();

然后按照vtk的可视化管线流程，Reader -> Mapper -> Actor，

// 3. 构建核心渲染管线 (Reader -> Mapper -> Actor)
const reader = vtkSTLReader.newInstance();//reader实例，负责搬运模型数据，变成程序可以识别的内存数据
const mapper = vtkMapper.newInstance();//mapper实例，负责给reader拿到的数据附上属性，比如颜色
const actor = vtkActor.newInstance();//数据实体，没有它模型只是一堆数据

// 组装管线
mapper.setInputConnection(reader.getOutputPort());
actor.setMapper(mapper);

//关闭mapper对模型的颜色改变
mapper.setScalarVisibility(false);

这里为什么要关闭mapper对模型的颜色改变呢，因为 Mapper 默认是个工作狂，只要它发现模型身上有数据，它就会强行按照数据去填充模型，因为我们需要先将模型统一涂成灰色，所以先让mapper暂停对模型的颜色改变。之后我们对模型的实例对象进行操作，给管道附上初始颜色，

// 获取 Actor 的属性存入property变量
const property = actor.getProperty();

// 自定义管道初始颜色
const grayColor = [0.70, 0.70, 0.75];
actor.getProperty().setColor(…grayColor);
property.setAmbientColor(…grayColor);
property.setDiffuseColor(…grayColor);

调整好场景的光照系数，让模型看起来更立体，

//调整场景光照系数
property.setAmbient(0.4); // 增加基础环境亮度 (01)，防止模型在暗处死黑
property.setDiffuse(0.9); // 满格的漫反射，让模型受灯光影响更明显
property.setSpecular(0.1); // 加上一点微弱的高光 (01)，产生一种轻微的金属质感
property.setSpecularPower(50); // 高光聚焦度，数值越大越像陶瓷，越小越像磨砂

随后，我们就可以加载模型了，

//加载 STL 模型
const stlFileName = ‘tisu6.stl’;
reader.setUrl(stlFileName).then(() => {
….//加载模型后应该执行的程序
}).catch((err) => {
console.error(“模型加载失败！”, err);
});

这里的 ‘tisu6.stl’是stl模型的文件名，注意要将模型文件放入public文件夹，另一个注意点就是我们reader.setUrl(stlFileName).then(() 这个语句，一定要保证模型数据加载完毕后才执行后面的程序，不然程序如果加载不完全就执行后面的程序的话，会出现难以预测的错误，这个语句的then就是为了保证模型加载完毕而存在的。
加载完程序后，将模型的多边形数据存入polydata中，方便之后操作，

const polydata = reader.getOutputData();

我们可以打印一下模型点数，看看模型一共有多少的点，

const numPoints = polydata.getNumberOfPoints();
console.log(>>> 模型加载成功！共有 ${numPoints} 个顶点。 <<<);

之前，我们定义了管道初始颜色为灰色，这里设置初始温度为20度，这样就把管道的初始温度20度和初始颜色灰色对应起来了，

// 初始化管道温度为20度
const tempValues = new Float32Array(numPoints);
tempValues.fill(20.0);
const temperatures = vtkDataArray.newInstance({
name: ‘Temperature’,
values: tempValues,
});
polydata.getPointData().setScalars(temperatures);

初始温度和颜色对应好之后，我们加下来配置好20度到100度对应的颜色，这里我们创建色带，将颜色与温度结合，为了保证颜色变化符合预期，我们采用插入锚点的方式，

// 色带创建，将颜色与温度结合，各个温度之间插入锚点，让颜色更符合预期
const lut = vtkColorTransferFunction.newInstance();
lut.addRGBPoint(20.0, 0.70, 0.70, 0.75);//20度到30度统一保持灰色
lut.addRGBPoint(30.0, 0.70, 0.70, 0.75);

lut.addRGBPoint(30.1, 0.0, 0.3, 1.0); // 锚点 1：30.1 度 - 极致亮蓝
lut.addRGBPoint(45.0, 0.0, 1.0, 1.0); // 锚点 2：45.0 度 - 鲜艳青色
lut.addRGBPoint(60.0, 0.0, 1.0, 0.0);// 锚点 3：60.0 度 - 极致鲜绿
lut.addRGBPoint(75.0, 1.0, 1.0, 0.0);// 锚点 4：75.0 度 - 极致鲜黄
lut.addRGBPoint(90.0, 1.0, 0.6, 0.0); // 锚点 5：90.0 度 - 极致鲜橙
lut.addRGBPoint(100.0, 1.0, 0.0, 0.0); // 锚点 6：100.0 度 - 极致鲜红

色带创建好之后，每当管道温度改变，我们都要重新配置管道颜色，于是我们重新打开之前关闭的mapper，并让mapper渲染的颜色与色带对应，

// 3：重新配置 Mapper让其给管道按色带染色
mapper.setLookupTable(lut);
mapper.setScalarRange(20.0, 100.0);
mapper.setScalarVisibility(true);
mapper.setColorModeToMapScalars();
mapper.setScalarModeToUsePointData();

// 将模型加入场景并渲染
renderer.addActor(actor);
renderer.resetCamera();
renderWindow.render();

现在模型已经在场景中搭好了，那么接下来我们给模型定义两种接收温度信息的方式，

//第一种是像这样用一个数组来更新温度，每次接收新的温度信息就改变数组对应参数
// 设置红外相机拍摄模型的点位位置和温度
const irTargets = [
{ id: ‘IR_POINT_01’, x: -753.3, y: 285.2, z: -96.1, tempBase: 30 },
{ id: ‘IR_POINT_02’, x: -753.6, y: 252.9, z: -96.8, tempBase: 45 },
{ id: ‘IR_POINT_03’, x: -753.3, y: 217.1, z: -96.1, tempBase: 55 },
….//点位数量可自定
];

//第二种方式则是用data.json文件来接收温度数据，每当文件内温度数据改变，管道上对应点位温度也随之改变
// 获取所有顶点的真实 XYZ 坐标数组
const pointsData = polydata.getPoints().getData();
//用一个数组存点位数据
let activeSources = irTargets.map(target => ({
x: target.x,
y: target.y,
z: target.z,
temp: target.tempBase // 初始用默认温度
}));

// 每5秒负责更新数据，数据来源data.json文件，每5s更新一次
setInterval(async () => {
let dynamicJsonArray = [];
try {
const response = await fetch(‘/data.json?t=’ + new Date().getTime());
if (response.ok) {
dynamicJsonArray = await response.json();
}
} catch (error) {
// 静默失败
}

activeSources = irTargets.map((target) => {
  const dataItem = dynamicJsonArray.find(item => item.id === target.id);
  let currentTemp;
  if (dataItem && dataItem.temperature !== undefined) {
    currentTemp = dataItem.temperature;
  } else {
    currentTemp = target.tempBase + Math.random() * 10;
  }
  return {
    x: target.x,
    y: target.y,
    z: target.z,
    temp: currentTemp 
  };
});

}, 5000); //每5s执行一次

为了让管道上温度可视化效果符合热力图变化，采用IDW（反距离权重插值法）算法，在一定范围内找到距离最近的温度点，自动计算两点之间的温度 ，实现温度的渐变效果，为了模拟真实场景下温度的自然冷却效果，我们可以在每次刷新管道温度数据时，让管道上的温度下降两度，且不低于环境温度，这两个功能随着模型刷新同步执行，模型每刷新一次，两个功能就执行一次，

// 每1.5秒重新渲染模型
setInterval(() => {
let needsRender = false; 

//全局自然冷却特效，模拟自然冷却效果，等待新数据时每此温度下降2度，最低降到环境温20度
const ambientTemp = 20.0;
const decayRate = 2.0; 

for (let i = 0; i < numPoints; i++) {
  if (tempValues[i] > ambientTemp) {
    tempValues[i] -= decayRate;
    if (tempValues[i] < ambientTemp) tempValues[i] = ambientTemp;
    needsRender = true;
  }
}

//IDW算法（反距离权重插值法），在一定范围内找到距离最近的温度点，自动计算两点之间的温度 
const maxInfluenceRadius = 45.0; 
const maxRadius2 = maxInfluenceRadius * maxInfluenceRadius;

for (let i = 0; i < numPoints; i++) {
  const px = pointsData[i * 3];
  const py = pointsData[i * 3 + 1];
  const pz = pointsData[i * 3 + 2];

  let numerator = 0.0;
  let denominator = 0.0;
  let minDist2 = maxRadius2;

  for (let s = 0; s < activeSources.length; s++) {
    const src = activeSources[s];
    
    if (Math.abs(px - src.x) > maxInfluenceRadius || 
        Math.abs(py - src.y) > maxInfluenceRadius || 
        Math.abs(pz - src.z) > maxInfluenceRadius) {
      continue; 
    }

    const dx = px - src.x;
    const dy = py - src.y;
    const dz = pz - src.z;
    const dist2 = dx * dx + dy * dy + dz * dz;

    if (dist2 < 1.0) { 
      tempValues[i] = src.temp;
      break;
    }

    if (dist2 < maxRadius2) { 
      const w = 1.0 / dist2; 
      numerator += src.temp * w;
      denominator += w;
      if (dist2 < minDist2) minDist2 = dist2;
    }
  }

  if (denominator > 0.0) {
    const interpolatedTemp = numerator / denominator;
    const edgeWeight = Math.max(0, 1.0 - (minDist2 / maxRadius2));
    const finalTemp = ambientTemp + (interpolatedTemp - ambientTemp) * edgeWeight;

    if (finalTemp > tempValues[i]) {
      tempValues[i] = finalTemp;
    }
    needsRender = true;
  }
}

// 推送到显卡，刷新模型
if (needsRender) {
  temperatures.modified(); 
  polydata.getPointData().modified();
  renderWindow.render();
}

}, 1500);

到这一步，我们红外相机温度可视化的基本功能就实现了，接下来，为了更好的视觉和使用体验，我们给程序加上另外一个功能，一个是鼠标悬停拾取器，基于Raycasting(射线投射)的3D鼠标拾取器，鼠标悬停在管道任意位置时，系统可穿透空间，精准命中管道物理表面，并在屏幕上实时显示该点三维坐标(X,Y,Z)及当前温度，温度超过一定值触发视觉报警，

// 鼠标悬停拾取器 (用于测量鼠标指向的点的坐标与温度)
const probeTooltip = document.createElement(‘div’);
probeTooltip.className = ‘probe-tooltip’;
document.body.appendChild(probeTooltip);
probeTooltip.style.pointerEvents = ‘none’;
const pointPicker = vtkPointPicker.newInstance();
pointPicker.setPickFromList(1); // 开启白名单模式
pointPicker.initializePickList();
pointPicker.addPickList(actor); // 只拾取管子，忽略空气
pointPicker.setTolerance(0.005);
const interactor = renderWindow.getInteractor();
interactor.onMouseMove((callData) => {
if (!callData.pokedRenderer || renderer !== callData.pokedRenderer) {
probeTooltip.style.display = ‘none’;
return;
}
const { x, y } = callData.position;
pointPicker.pick([x, y, 0], renderer);
const pickedActors = pointPicker.getActors();
if (pickedActors.length > 0) {
const pointId = pointPicker.getPointId();
if (pointId !== undefined && pointId !== -1) {
const temp = tempValues[pointId];
const px = pointsData[pointId * 3];
const py = pointsData[pointId * 3 + 1];
const pz = pointsData[pointId * 3 + 2];
probeTooltip.innerHTML = ID: ${pointId}<br/>Temp: ${temp.toFixed(1)} °C<br/>Pos: (${px.toFixed(1)}, ${py.toFixed(1)}, ${pz.toFixed(1)});

    // 温度报警变色逻辑
    if (temp > 60.0) {
      probeTooltip.style.color = '#ff3333';
      probeTooltip.style.borderColor = '#ff3333';
      probeTooltip.style.boxShadow = '0 0 10px rgba(255, 50, 50, 0.4)';
    } else {
      probeTooltip.style.color = '#00ffcc';
      probeTooltip.style.borderColor = '#00ffcc';
      probeTooltip.style.boxShadow = 'none';
    }
    const dpr = window.devicePixelRatio || 1; 
    const cssX = x / dpr;
    const cssY = y / dpr;
    const screenY = window.innerHeight - cssY;
    probeTooltip.style.left = `${cssX + 20}px`;
    probeTooltip.style.top = `${screenY - 80}px`;
    
    // 显示标签
    probeTooltip.style.display = 'block';
  } else {
    probeTooltip.style.display = 'none';
  }
} else {
  probeTooltip.style.display = 'none';
}

});

//当鼠标彻底离开网页区域时，强制隐藏标签
document.addEventListener(‘mouseleave’, () => {
probeTooltip.style.display = ‘none’;
});

// 创建悬浮框，根据鼠标指向实时显示点位信息（温度，坐标，ID）
const legendContainer = document.createElement(‘div’);
legendContainer.className = ‘scada-legend’;
// 直接用 HTML 注入标题、色块和刻度文字
legendContainer.innerHTML = <div class="legend-title">Temp (C)</div> <div class="legend-content"> <div class="legend-color-bar"></div> <div class="legend-ticks"> <span>100.0</span> <span>80.0</span> <span>60.0</span> <span>40.0</span> <span>20.0</span> </div> </div> ;
document.body.appendChild(legendContainer);
